Soil Type

The Soil Type tab allows for the definition of soil types used in the Soil Layer page. For each soil type the following parameters can be defined:

  • Name
  • Unit weight
  • Initial damping
  • Normalized shear modulus (G/Gmax) degradation model
  • Damping model
  • Notes

Defining Standard Deviation Models

The uncertainty of the non-linear curves is characterized by a truncated normal distribution. The distribution is truncated with a minimum value to prevent errorneous negative values. UTSR allows the user flexibility in the definition of the standard deviation. In the simplest manner, a constant standard deviation can defined. However, the user also has access to the strain and magnitude of the property, and a variety of mathimatical functions.

In the formula for the standard deviation of the shear modulus, the following variables can be used:

  • strain -- shear strain in percent
  • shearMod -- normalized shear modulus
Likewise, the following variables are available in the definition of the standard deviation of the damping:
  • strain -- shear strain in percent
  • damping -- damping in decimal
The following functions are available:
  • abs(x)
  • acos(x)
  • asin(x)
  • atan(x)
  • atan2(y, x)
  • ceil(x)
  • cos(x)
  • exp(x)
  • floor(x)
  • log(x)
  • max([value1 [, value2 [, ...]]])
  • min([value1 [, value2 [, ...]]])
  • pow(x, y)
  • random()
  • round(x)
  • sin(x)
  • sqrt(x)
  • tan(x)
